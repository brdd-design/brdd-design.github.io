# 🌐 BRDD Official Portal

This repository contains the source code for the official **Business Rule Driven Design (BRDD)** knowledge hub.

## 🚀 Technologies
- **Vite:** Next-generation frontend tooling.
- **Vanilla JS / CSS:** Maximum performance and zero dependencies.
- **i18n:** Built-in internationalization support (EN/PT-BR).

## 🛠 Development
To run the project locally:
```bash
make dev
```
